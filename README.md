# Open Browser

An open source browser infra to spin up thousands of concurrent browsers without any limitations.

## Why am I building this?

There ARE buncha brainded browser infra companies nowadays advertising for agents. BUT I believe agents don't need such bloated browsers. What agents really want is the power to run MULTIPLE parallel browsers at once.

## My mission

- Worked on browser infra for the past year
- Make deploying and accessing browsers cheap and easy!
- Infra deployable in a single command
- Just doing this for the love of the game
