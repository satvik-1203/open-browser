# Open Browser

An open source browser infra to spin up thousands of concurrent browsers without any limitations.

## Why am I building this?

I Worked on browser infra for the past year, and I beleive there ARE buncha browser infra companies nowadays advertising for agents. However, agents don't need such bloated metal browsers but require the ability to run MULTIPLE parallel browsers at once.

## My mission
- Make deploying and accessing browsers cheap and easy!
- Infra deployable in a single command
- Just doing this for the love of the game
